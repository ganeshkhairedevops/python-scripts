# Python Scripts for DevOps 🚀

This repository contains practical Python scripts focused on automation, monitoring, and DevOps workflows.

The goal is to use Python for solving real-world DevOps problems — not just basic programming exercises.

---

## 📂 Repository Structure

### Server Health Monitor
- CPU, memory, disk checks
- Status-based reporting
- Beginner-friendly automation

### Log Analyzer
- Parse application logs
- Count errors and warnings
- Filter logs by keyword
